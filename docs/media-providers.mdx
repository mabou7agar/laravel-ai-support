# Media Providers

Laravel AI Engine supports hosted and local media generation providers for image, video, audio generation, and transcription.

## Providers

| Provider | Engine | Best for | Key env |
| --- | --- | --- | --- |
| OpenAI audio | `openai` | TTS and transcription beside chat/agent workflows | `OPENAI_API_KEY` |
| Cloudflare Workers AI | `cloudflare_workers_ai` | low-cost image, TTS, transcription | `CLOUDFLARE_API_TOKEN`, `CLOUDFLARE_ACCOUNT_ID` |
| Hugging Face Inference Providers | `huggingface` | provider marketplace for image/audio models | `HUGGINGFACE_API_KEY` or `HF_TOKEN` |
| Replicate | `replicate` | public image/video/audio model catalog | `REPLICATE_API_TOKEN` |
| ComfyUI | `comfyui` | local/self-hosted image and video workflows | `COMFYUI_BASE_URL` |
| Gemini media | `gemini` | Imagen, Veo, native TTS, and Lyria models | `GEMINI_API_KEY` |
| Google Text-to-Speech | `google_tts` | Google Cloud TTS voices | `GOOGLE_TTS_API_KEY` or `GOOGLE_TTS_ACCESS_TOKEN` |
| ElevenLabs | `eleven_labs` | high-quality voices and narration | `ELEVENLABS_API_KEY` |
| FAL | `fal_ai` | async video and catalog execution | `FAL_AI_API_KEY` |
| Pexels | `pexels` | stock photo search and curated image discovery | `PEXELS_API_KEY` |

## Sync Models

Register the low-cost media providers into the `ai_models` table:

```bash
php artisan ai:sync-models --provider=media
```

You can sync one provider:

```bash
php artisan ai:sync-models --provider=cloudflare_workers_ai
php artisan ai:sync-models --provider=huggingface
php artisan ai:sync-models --provider=replicate
php artisan ai:sync-models --provider=comfyui
```

## Text To Speech

OpenAI TTS is available through the normal OpenAI driver:

```php
$response = app(\LaravelAIEngine\Services\AIEngineService::class)->generateAudio(
    new \LaravelAIEngine\DTOs\AIRequest(
        prompt: 'Read this invoice summary.',
        engine: 'openai',
        model: 'gpt-4o-mini-tts',
        parameters: [
            'voice' => 'nova',
            'format' => 'mp3',
        ],
    )
);
```

Google Cloud Text-to-Speech is available through the dedicated `google_tts` engine:

```php
$response = app(\LaravelAIEngine\Services\AIEngineService::class)->generateAudio(
    new \LaravelAIEngine\DTOs\AIRequest(
        prompt: 'Read this customer note.',
        engine: 'google_tts',
        model: 'google-tts',
        parameters: [
            'voice' => 'en-US-Neural2-F',
            'language_code' => 'en-US',
            'audio_encoding' => 'MP3',
        ],
    )
);
```

Gemini native text-to-speech is available through the normal Gemini driver. The package calls `generateContent` with audio response modalities, extracts inline PCM audio, wraps it as WAV, and stores it through the configured media disk:

```php
$response = app(\LaravelAIEngine\Services\AIEngineService::class)->generateAudio(
    new \LaravelAIEngine\DTOs\AIRequest(
        prompt: 'Read this invoice summary.',
        engine: 'gemini',
        model: 'gemini-2.5-flash-preview-tts',
        parameters: [
            'voice' => 'Kore',
        ],
    )
);
```

For multi-speaker TTS, pass Gemini speech config directly:

```php
new \LaravelAIEngine\DTOs\AIRequest(
    prompt: 'Speaker 1: Hello. Speaker 2: Welcome.',
    engine: 'gemini',
    model: 'gemini-2.5-flash-preview-tts',
    parameters: [
        'speaker_voice_configs' => [
            ['speaker' => 'Speaker 1', 'voiceConfig' => ['prebuiltVoiceConfig' => ['voiceName' => 'Kore']]],
            ['speaker' => 'Speaker 2', 'voiceConfig' => ['prebuiltVoiceConfig' => ['voiceName' => 'Puck']]],
        ],
    ],
);
```

`lyria-002` remains available for music generation style workflows. Media routing now defaults Gemini `audio_generation` to native TTS and keeps Lyria under `music_generation`.

## Pexels Stock Photos

Pexels is a stock-photo search provider, not an LLM. Use it when the application needs licensed photo results instead of generated images:

```php
$response = app(\LaravelAIEngine\Services\AIEngineService::class)->generateImage(
    new \LaravelAIEngine\DTOs\AIRequest(
        prompt: 'laptop on office desk',
        engine: 'pexels',
        model: 'pexels-search',
        parameters: [
            'per_page' => 10,
            'orientation' => 'landscape',
        ],
    )
);
```

The response content is a JSON array of normalized photo results with URLs, dimensions, color metadata, photographer attribution, and the original Pexels URL.

## Cost-Aware Routing

Use `LaravelAIEngine\Services\Media\MediaProviderRouter` to choose a provider by capability:

```php
$selection = app(\LaravelAIEngine\Services\Media\MediaProviderRouter::class)
    ->select('image', 'cheapest');

// ['provider' => 'cloudflare_workers_ai', 'model' => '@cf/black-forest-labs/flux-1-schnell', ...]
```

Supported routing modes:

| Mode | Behavior |
| --- | --- |
| `cheapest` | Lowest configured estimated unit cost |
| `local` / `free` | Prefers local/free models such as ComfyUI |
| `fast` | Prefers lower configured latency score |
| `quality` | Prefers higher configured quality score |
| `balanced` | Cost divided by quality score |

Configure estimates in `ai-engine.media_routing.providers`. These are app-side estimates for routing and credit policy; provider invoices remain the source of truth.

Provider routing skips disabled providers, providers with missing configured credentials, and providers marked unhealthy until a future timestamp:

```php
'media_routing' => [
    'providers' => [
        'replicate' => [
            'enabled' => true,
            'api_key_config' => 'ai-engine.engines.replicate.api_key',
            'unhealthy_until' => null,
            'models' => [
                'image' => [
                    'model' => 'black-forest-labs/flux-schnell',
                    'estimated_unit_cost' => 0.004,
                    'quality_score' => 1.7,
                    'latency_score' => 4.0,
                ],
            ],
        ],
    ],
],
```

Use `constraints: ['allow_local' => false]` when an application must avoid local/self-hosted providers such as ComfyUI.

## ComfyUI Workflows

ComfyUI needs a workflow graph. Configure a default workflow:

```php
'engines' => [
    'comfyui' => [
        'base_url' => env('COMFYUI_BASE_URL', 'http://127.0.0.1:8188'),
        'default_workflow' => [
            // ComfyUI workflow JSON. Use {{prompt}} where the user prompt should be inserted.
        ],
    ],
],
```

You can also pass a workflow per request:

```php
new \LaravelAIEngine\DTOs\AIRequest(
    prompt: 'product photo on white background',
    engine: 'comfyui',
    model: 'comfyui/default-image',
    parameters: ['workflow' => $workflow]
);
```

## Notes

- Cloudflare image/audio endpoints use the Workers AI REST model runner.
- Hugging Face is intentionally generic; pass `provider` or provider-specific parameters when needed.
- Replicate supports model endpoints and versioned predictions.
- Pexels returns existing stock-photo results; it does not generate images or support streaming.
- Video generation should usually be queued or async in application code because provider jobs can run longer than HTTP request timeouts.
- Stable enum aliases are for built-in defaults. For changing provider catalogs, sync models into `ai_models` and let the dynamic model resolver route by provider/model id.
- Gemini inline TTS audio is stored as WAV so browser and app clients receive a normal downloadable file instead of raw PCM bytes.
