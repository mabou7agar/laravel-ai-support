# Realtime And Observability

Realtime descriptors, tool brokers, traces, and evaluations let host apps connect provider realtime clients without moving application policy out of Laravel.

## Realtime Sessions

```php
use LaravelAIEngine\DTOs\RealtimeSessionConfig;
use LaravelAIEngine\Facades\Engine;

$session = Engine::realtime()->create(
    RealtimeSessionConfig::make('openai', 'gpt-4o-realtime-preview')
        ->withVoice('alloy')
        ->withAudioFormats('pcm16', 'pcm16')
        ->withTurnDetection(['type' => 'server_vad'])
        ->withTemperature(0.2)
        ->withMaxResponseOutputTokens(512)
);
```

Realtime support returns provider-specific session descriptors so host applications can attach their own WebSocket/WebRTC clients. OpenAI descriptors use snake_case fields expected by Realtime APIs. Gemini Live descriptors use Gemini-style camelCase fields such as `responseModalities`, `systemInstruction`, and `generationConfig`.

## Durable Agent Run Streams

Agent chat can create durable queued runs by sending `async=true` to `POST /api/v1/agent/chat`. The response includes `status_url`, `trace_url`, `stream_url`, and Laravel broadcast metadata.

Use SSE when a browser tab needs live progress without a WebSocket service:

```js
const events = new EventSource(`/api/v1/ai/agent-runs/${runId}/stream?timeout=30&poll=500`);

events.addEventListener('run.completed', (event) => {
  const payload = JSON.parse(event.data);
  console.log(payload.payload);
  events.close();
});

events.addEventListener('run.failed', () => events.close());
```

Enable Laravel Broadcasting when the app already uses Reverb, Pusher, Soketi, Ably, or another broadcast driver:

```env
AI_AGENT_EVENT_STREAM_BROADCAST_ENABLED=true
AI_AGENT_EVENT_STREAM_BROADCAST_PRIVATE=true
AI_AGENT_EVENT_STREAM_BROADCAST_CHANNEL_PREFIX=agent-run
```

```js
Echo.private(`agent-run.${runId}`)
  .listen('.tool.started', (event) => console.log(event))
  .listen('.approval.required', (event) => console.log(event))
  .listen('.artifact.created', (event) => console.log(event))
  .listen('.run.completed', (event) => console.log(event));
```

Polling remains available through `GET /api/v1/ai/agent-runs/{run}` and `GET /api/v1/ai/agent-runs/{run}/trace`; both return persisted fallback events.

## Realtime Tool Broker

`LaravelAIEngine\Services\SDK\RealtimeToolBrokerService` dispatches provider realtime tool calls through the package `ToolRegistry` and returns a structured result payload for the host client to send back to the provider.

Tools that require confirmation return `approval_required` instead of executing.

```bash
curl -X POST http://localhost:8000/api/v1/ai/realtime/tools/dispatch \
  -H "Accept: application/json" \
  -H "Content-Type: application/json" \
  -d '{
    "event": {"id": "call_1", "name": "run_skill", "arguments": {"message": "Draft a reply"}},
    "session_id": "support-thread-123",
    "user_id": "user-1",
    "approved": false
  }'
```

## MCP/App Bridge

`LaravelAIEngine\Services\SDK\McpAppToolAdapter` exports registered tools and enabled skills in an MCP/App-compatible schema and can call local tools through the same registry.

```bash
curl -H "Accept: application/json" \
  http://localhost:8000/api/v1/ai/mcp/tools

curl -X POST http://localhost:8000/api/v1/ai/mcp/tools/run_skill/call \
  -H "Accept: application/json" \
  -H "Content-Type: application/json" \
  -d '{
    "arguments": {"message": "Draft a reply"},
    "session_id": "support-thread-123",
    "user_id": "user-1"
  }'
```

## Tracing And Evaluations

```php
$span = Engine::traces()->start('support-agent.run');
Engine::traces()->end($span, metadata: ['tokens' => 420]);

Engine::evaluations()->assertPassed(
    'contains source',
    'Answer [1]',
    assertion: fn (string $answer) => str_contains($answer, '[')
);
```

Register exporters in `ai-engine.observability.exporters`:

```php
'observability' => [
    'exporters' => [
        LaravelAIEngine\Services\SDK\Exporters\HttpObservabilityExporter::class,
        LaravelAIEngine\Services\SDK\Exporters\OpenTelemetryObservabilityExporter::class,
        LaravelAIEngine\Services\SDK\Exporters\LangSmithObservabilityExporter::class,
        LaravelAIEngine\Services\SDK\Exporters\LogObservabilityExporter::class,
    ],
    'http' => [
        'endpoint' => env('AI_ENGINE_OBSERVABILITY_HTTP_ENDPOINT'),
        'headers' => ['Authorization' => 'Bearer '.env('OBSERVABILITY_TOKEN')],
    ],
],
```
