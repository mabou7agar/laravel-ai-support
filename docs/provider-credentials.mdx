# Provider Credentials

You do not need every provider token to use Laravel AI Engine. Start with the providers your application actually needs, then add optional providers for cost, fallback, local execution, or specialized media.

## Recommended Starting Set

| Capability | Provider | Env |
| --- | --- | --- |
| Chat, agents, tool calling, structured output, hosted files | OpenAI | `OPENAI_API_KEY` |
| Image, video, and audio generation catalog | FAL | `FAL_AI_API_KEY` |
| Semantic memory, RAG, nearest tool and skill retrieval | Qdrant | `QDRANT_API_KEY` |
| Entity relationships, permissions, and graph memory | Neo4j | `NEO4J_URI`, `NEO4J_USERNAME`, `NEO4J_PASSWORD` |
| Local/private development | Ollama, ComfyUI | `OLLAMA_BASE_URL`, `COMFYUI_BASE_URL` |

## Provider Matrix

| Provider | When to use | Required env |
| --- | --- | --- |
| OpenAI | Main production chat, agents, tools, vision, structured JSON, hosted files, and vector workflows. | `OPENAI_API_KEY` |
| Anthropic | Reasoning-heavy tasks, long document analysis, planning, or fallback from OpenAI. | `ANTHROPIC_API_KEY` |
| Gemini | Cost-sensitive multimodal work, large context, Imagen, Veo, and Lyria media fallback. | `GEMINI_API_KEY` |
| FAL | Production image, video, audio generation, async media jobs, and dynamic catalog execution. | `FAL_AI_API_KEY` |
| Cloudflare Workers AI | Low-cost edge inference, lightweight image/audio/transcription, and Cloudflare-hosted apps. | `CLOUDFLARE_API_TOKEN`, `CLOUDFLARE_ACCOUNT_ID` |
| Hugging Face | Open model experimentation, specialized image/audio models, and provider marketplace routing. | `HUGGINGFACE_API_KEY` or `HF_TOKEN` |
| Replicate | Public image/video/audio model endpoints and quick access to community models. | `REPLICATE_API_TOKEN` |
| ComfyUI | Local or self-hosted image/video workflows, custom node graphs, and low provider cost. | `COMFYUI_BASE_URL` |
| Stability AI | Stability image models when you need that provider directly. | `STABILITY_API_KEY` |
| ElevenLabs | High-quality text-to-speech, voices, narration, and agent audio. | `ELEVENLABS_API_KEY` |
| OpenRouter | Router across many LLMs with one API key, model comparison, and fallback. | `OPENROUTER_API_KEY` |
| NVIDIA NIM | Enterprise or GPU-hosted model endpoints. | `NVIDIA_NIM_API_KEY` or `NVIDIA_API_KEY` |
| Ollama | Local/offline chat and embedding development. | `OLLAMA_BASE_URL` |
| Qdrant | Semantic search, scoped RAG, memory, and nearest tool or skill discovery. | `QDRANT_API_KEY` |
| Pinecone | Managed vector database alternative when your application already uses Pinecone. | `PINECONE_API_KEY` |
| Neo4j | Graph memory, linked entities, tenant/user/workspace relationships, and explainable links. | `NEO4J_URI`, `NEO4J_USERNAME`, `NEO4J_PASSWORD` |
| LangGraph | Optional durable agent graph runtime for resumable workflows and human approval loops. | `AI_AGENT_LANGGRAPH_API_TOKEN` |

## Cost And Markup

Provider rate multipliers let you charge more than the raw provider cost without changing driver code:

```env
AI_OPENAI_RATE=2.0
AI_ANTHROPIC_RATE=1.5
AI_GEMINI_RATE=1.2
AI_FAL_AI_RATE=1.3
AI_CLOUDFLARE_WORKERS_AI_RATE=1.2
AI_HUGGINGFACE_RATE=1.2
AI_REPLICATE_RATE=1.25
AI_COMFYUI_RATE=1.0
AI_OPENROUTER_RATE=1.25
AI_NVIDIA_NIM_RATE=1.2
```

Media routing estimates live in `ai-engine.media_routing.providers`. These estimates are used for app-side routing and credit policy only; provider invoices remain the source of truth.

## Practical Defaults

For most applications, use OpenAI for chat and tool use, Qdrant for semantic retrieval, Neo4j for linked business entities, and FAL or Gemini for hosted media. Add Cloudflare, Hugging Face, Replicate, Ollama, or ComfyUI when you need lower cost, local execution, or model variety.
