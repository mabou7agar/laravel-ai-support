## Package API Middleware Injection

```env
# append to all package API groups
AI_ENGINE_API_APPEND_MIDDLEWARE=

# append to specific groups
AI_ENGINE_API_RAG_MIDDLEWARE=
AI_ENGINE_API_ACTIONS_MIDDLEWARE=
AI_ENGINE_API_MODULES_MIDDLEWARE=
AI_ENGINE_API_GENERATE_MIDDLEWARE=
AI_ENGINE_API_DATA_COLLECTOR_MIDDLEWARE=
AI_ENGINE_API_AUTONOMOUS_COLLECTOR_MIDDLEWARE=
AI_ENGINE_API_DEMO_MIDDLEWARE=
```

Example:

```env
AI_ENGINE_API_APPEND_MIDDLEWARE=auth:sanctum
AI_ENGINE_API_GENERATE_MIDDLEWARE=throttle:30,1
# Multiple middlewares: use semicolon separator
# AI_ENGINE_API_GENERATE_MIDDLEWARE=auth:sanctum;throttle:30,1
# Exact syntax also supports JSON arrays:
# AI_ENGINE_API_GENERATE_MIDDLEWARE=["auth:sanctum","throttle:30,1"]
```
