## Engine Defaults

```env
AI_ENGINE_DEFAULT=openai
AI_ENGINE_DEFAULT_MODEL=gpt-4o
AI_ORCHESTRATION_MODEL=gpt-4o-mini
AI_ENGINE_REQUEST_PROVIDER_PRIORITY=native;openrouter;anthropic;gemini;deepseek;nvidia_nim;ollama
```

`AI_ENGINE_REQUEST_PROVIDER_PRIORITY` controls provider resolution for direct requests when callers omit `engine`. The special `native` token means "prefer the model's direct provider first", then try the rest in order.


Neo4j-first defaults do not force Neo4j at runtime when credentials are missing. In that case the package falls back to the configured vector driver, which is still `qdrant` by default.

`AI_ENGINE_NEO4J_VECTOR_NAMING_STRATEGY` supports:

- `static`
- `node`
- `tenant`
- `node_tenant`

Use `php artisan ai:backend-status` to confirm the effective read backend and whether fallback is active.

## Package API Middleware Injection

```env
# append to all package API groups
AI_ENGINE_API_APPEND_MIDDLEWARE=

# append to specific groups
AI_ENGINE_API_RAG_MIDDLEWARE=
AI_ENGINE_API_ACTIONS_MIDDLEWARE=
AI_ENGINE_API_MODULES_MIDDLEWARE=
AI_ENGINE_API_GENERATE_MIDDLEWARE=
AI_ENGINE_API_DATA_COLLECTOR_MIDDLEWARE=
AI_ENGINE_API_AUTONOMOUS_COLLECTOR_MIDDLEWARE=
```

Example:

```env
AI_ENGINE_API_APPEND_MIDDLEWARE=auth:sanctum
AI_ENGINE_API_GENERATE_MIDDLEWARE=throttle:30,1
# Multiple middlewares: use semicolon separator
# AI_ENGINE_API_GENERATE_MIDDLEWARE=auth:sanctum;throttle:30,1
# Exact syntax also supports JSON arrays:
# AI_ENGINE_API_GENERATE_MIDDLEWARE=["auth:sanctum","throttle:30,1"]
```

## Package Generic Defaults

```env
AI_ENGINE_DEMO_USER_ID=
AI_ENGINE_ENABLE_DEMO_ROUTES=false
AI_ENGINE_AUTH_ROUTES_ENABLED=false
```

Demo routes and package auth helper routes are outside the package core. Host applications should pass authenticated user, tenant, workspace, and other scope values through request payloads, `AIScopeOptionsService`, middleware, or model config hooks instead of relying on package fallbacks.

## Agent Conversation Context

```env
AI_AGENT_CONTEXT_COMPACTION_ENABLED=true
AI_AGENT_CONTEXT_MAX_MESSAGES=12
AI_AGENT_CONTEXT_KEEP_RECENT_MESSAGES=6
AI_AGENT_CONTEXT_MAX_MESSAGE_CHARS=2000
AI_AGENT_CONTEXT_MAX_TOTAL_CHARS=12000
AI_AGENT_CONTEXT_MAX_SUMMARY_CHARS=4000
AI_AGENT_CONTEXT_SUMMARY_MESSAGE_CHARS=240
```

Context compaction keeps recent chat turns verbatim and stores older turns in `metadata.conversation_summary`. The summary is reused in routing, conversational replies, and RAG context so long user sessions do not overflow provider context limits.
